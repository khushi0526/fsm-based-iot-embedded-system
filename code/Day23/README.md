# Day 23 – ESP32 OLED Display System (Live Data Display)

## 📌 Objective

To display real-time data on an OLED screen using ESP32, creating a user interface for embedded systems.

---

## 🛠 Hardware Used

* ESP32
* OLED Display (SSD1306 – I2C)
* Breadboard
* Jumper Wires

---

## 🔌 Pin Configuration

### OLED Display (I2C)

VCC → 3.3V
GND → GND
SDA → GPIO 21
SCL → GPIO 22

---

## ⚙ Working Principle

1. ESP32 communicates with OLED using I2C protocol.
2. Data is sent from ESP32 to the display.
3. The OLED shows:

   * Text
   * Sensor values
   * System messages

---

## 🧠 Concepts Learned

* I2C communication
* Display interfacing
* Data visualization
* Embedded UI design
