# Day 1 – LED Blink using ESP32

## 📌 Objective

To understand basic GPIO output by blinking an LED using ESP32.
This is the first step in learning how microcontrollers control external hardware.

---

## 🛠 Hardware Used

* ESP32
* LED
* 220Ω Resistor
* Jumper Wires

---

## 🔌 Pin Configuration

* LED → GPIO 2 (OUTPUT)
* GND → GND

---

## ⚙ Working Principle

* The ESP32 controls the LED using a digital output pin.
* When the GPIO pin is set to **HIGH**, current flows and the LED turns **ON**.
* When the GPIO pin is set to **LOW**, current stops and the LED turns **OFF**.
* A delay is used to create the blinking effect.

---

## 🧠 Concepts Learned

* GPIO output control
* Digital HIGH and LOW signals
* Using `pinMode()` and `digitalWrite()`
* Using `delay()` for timing
